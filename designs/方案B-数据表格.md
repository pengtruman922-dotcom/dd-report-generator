# 方案B：数据表格布局

## 整体风格
以表格为核心，类似 Excel / Airtable 风格。适合批量管理和快速对比多份报告。信息密度最高。

## 表格列定义

| 列名 | 宽度 | 说明 |
|------|------|------|
| 复选框 | 40px | 批量操作选择 |
| 编号 | 100px | BD编号，可排序 |
| 公司名称 | 自适应 | 公司全称，点击进入报告 |
| 行业 | 120px | 行业标签 |
| 评分 | 80px | 五维度综合分(0-10)，颜色区分 |
| 评级 | 100px | 星级 + 文字(推荐/谨慎/不推荐) |
| 状态 | 100px | 完成/生成中/失败 |
| 营收 | 100px | 最新营收数据 |
| 净利 | 100px | 最新净利润 |
| 日期 | 80px | 生成日期 |
| 操作 | 100px | 查看/下载/删除图标按钮 |

## 评分颜色编码

```
8.0-10.0  → 🟢 深绿色背景  强烈推荐
6.5-7.9   → 🟢 浅绿色背景  推荐
5.0-6.4   → 🟡 黄色背景    谨慎推荐
3.5-4.9   → 🟠 橙色背景    不推荐
0.0-3.4   → 🔴 红色背景    不建议
```

---

## 筛选功能

- **状态筛选**：全部 / 已完成 / 生成中 / 失败
- **评级筛选**：全部 / 推荐及以上 / 谨慎 / 不推荐
- **行业筛选**：下拉选择（从已有报告中提取行业列表）
- **搜索**：公司名称、BD编号模糊匹配
- **列排序**：点击表头排序（编号、评分、营收、日期等）

---

## 批量操作

选中多项后顶部出现操作栏：
- **批量下载**：打包为 .zip 下载
- **批量删除**：确认后删除
- **全选/取消全选**

---

## 交互说明

1. **点击公司名称** → 进入报告详情页
2. **点击 👁** → 新标签页打开报告预览
3. **点击 ⬇** → 下载 .md 文件
4. **点击 🗑** → 确认删除弹窗
5. **生成中的行** → 显示内联进度条，不可选中
6. **表头点击** → 切换升序/降序排列
7. **分页** → 底部分页器，可选每页 10/20/50 条

---

## 优点
- 信息密度最高，适合管理大量报告（50+）
- 支持排序、筛选、批量操作，效率最高
- 类 Excel 体验，用户上手门槛低
- 方便对比不同公司的关键指标

## 缺点
- 视觉上偏素，缺少直观的视觉冲击
- 每行信息紧凑，需要仔细看
- 不如卡片布局那样一眼看到评级差异

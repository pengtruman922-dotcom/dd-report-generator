# 方案C：左右分栏面板

## 整体风格
左侧为报告列表（紧凑），右侧为报告预览面板。类似邮箱客户端（Outlook）或文档管理器风格。无需跳转页面即可预览报告内容。

## 分栏比例

- 左侧列表：**300px** 固定宽度
- 右侧预览：**剩余空间**自适应
- 中间分割线可拖拽调整（可选功能）

---

## 交互说明

1. **点击左侧列表项** → 右侧实时加载并预览该报告
2. **左侧搜索** → 即时过滤列表
3. **右侧操作按钮** → 下载/复制/打印/删除当前报告
4. **生成中的项** → 点击后右侧显示实时进度日志
5. **键盘导航** → 上下键切换选中报告
6. **"+ 新建报告"** → 跳转首页上传流程

---

## 优点
- 不离开页面即可预览完整报告，体验最流畅
- 左侧快速切换，右侧即时预览，对比效率高
- 适合需要反复查阅、对比多份报告的场景
- 类邮箱客户端，操作直觉性强

## 缺点
- 左侧列表宽度有限，公司名过长会截断
- 小屏幕上体验可能较差（需要做响应式处理）
- 同时只能看一份报告，不适合并排对比
